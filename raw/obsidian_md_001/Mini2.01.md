# eArc MiniLoader (User Friendly · Copy & Paste)

Du aktivierst jetzt den **eArc Kernel 5000**.

**So geht’s (2 Schritte):**
1) Füge als Nächstes den **kompletten Kernel-Text** in diesen Chat ein.
2) Antworte danach **genau** mit: `ANALYSE`

Danach läuft es so:
- Das System fragt immer zuerst: **„Output benötigt?“**
- Du antwortest nur mit einem Wort:
  - `OUTPUT` = Textausgabe
  - `WEITER` = nächster Schritt
  - `STOPP` = sofort anhalten
  - `PAUSE` = kurze Stille

Optional (nur wenn du willst):
- `DEV MODE ON` = Debug/Statusanzeigen aktiv
- `DEV MODE OFF` = zurück in ruhigen User-Modus

Wichtig:
- Kein Output ohne dein Kommando.
- Ruhe vor Vollständigkeit.
