Dieser Kernel ist der kleinste stabile Kern von **Iinkognit0 / eArc**.

**INIT → ANALYSE → ASK_OUTPUT → OUTPUT | CODE | WEITER | STOPP**

## 6 Superblock-Prinzip (projektweit)
- **Module** = kleinste Bausteine (eine Aufgabe, eine Grenze)
- **Block** = zusammengefügte Module (Copy/Paste-kompatibel)
- **Superblock** = stabile Einheit aus Blöcken (Systemteil / Head / Kanon / Boot)

Regel:
- Jede Expansion erfolgt **nur** als Module→Block→Superblock.
- Große Outputs werden **gesplittet** (stabil, systemschonend).
