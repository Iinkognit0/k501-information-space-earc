# eArc Kernel 5000 v2.6.0
## Das Ewige Archiv

## Module
### 1) eArcDynamicTextFlow (Core)
- Text als Primärmedium (Archiv-first)
- Dynamische Proportionen statt Templates
- Text entspringt aus der Mitte
- Adaptiver Satzspiegel (55–65 %)
- Einseiter-Modus (optional): keine Witwen/Waisen, kein Einzelwort

### 2) Voice Layer (Text-to-Speech)
- Optionaler Lesemodus, Silent Default
- One-Voice-Rule
- Tempo folgt Textstruktur
- Audio ist Ableitung, rebuildable, nie Ursprung
- Engine als Adapter (austauschbar)

### 3) Render Profiles
- PDF: gesetzt, kuratiert (Pandoc + LaTeX)
- EPUB: reflowable, leserzentriert (CSS)
- HTML: semantisch, ruhig, archivfähig

### 4) Optimierung
- Preset-Stufen (S/M/L/XL) statt Mikro-Tuning
- Samepage-Logik im Einseiter-Modus
- Konsistentes Fontset (Serif/Sans/Mono) vorgesehen

## Geltung
- Änderungen nur über neue Kernel-Version
- Alle Regeln und Module sind eingefroren
