## 1. Grundarchitektur

Das Frames Network ist kein klassisches Netzwerk, sondern ein
geordnetes Geflecht von Bedeutung.

- lokal entstehend
- verteilt haltbar
- wirksam nur durch bewusste Festlegung

Technik dient der Tragfähigkeit, nicht der Dynamik.

---

## 2. Frame-Ebene (kleinste Einheit)

### Zweck
- bewusster, abgeschlossener Zustand
- klein, adressierbar, archivfähig
- Ursprung jeder Bedeutung

### Struktur
- `frame.md` (Inhalt, Text-first)
- `meta.yaml` (ID, Status, Zeit, Herkunft)
- Status: drafted · fixed · archived
- Revision ergänzt, ersetzt nicht

### Tools / Bewertung
- Markdown + YAML → stabil, dauerhaft
- Hash (sha256) → Integrität
- Feed-Parser → optional (nur bei Import)
